Animations & UI Feedback
A single HTML page showing different Tailwind CSS animations and UI feedback elements.

Feachers
1. Built-in Animations – spin, ping, pulse, bounce
2. Button Animations – scale, shadow, color change, rotate, lift, combo
3. Card Animations – hover lift, group hover, border reveal
4. Notification Badges – bell, message, email icons with badges
5. Loading States – skeleton loader, spinner button, bouncing dots
6. Staggered Animations – cards fading in with different delays

How to run
1. Download the index.html file
2. Open it in any browser
